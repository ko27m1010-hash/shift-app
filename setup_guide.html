🌾 シフト希望Webアプリ セットアップガイド

このガイドに沿って設定すると、スマホアプリとして使えるようになります。

1ファイル構成

以下のファイルを同じフォルダに置いてください：

  • index.html — メインアプリ（スマホ最適化Webアプリ）
  • manifest.json — PWA設定（ホーム画面追加用）
  • sw.js — Service Worker（オフライン対応）
  • icon-192.png / icon-512.png — アプリアイコン
  • GAS_Webhook受信スクリプト.js — スプレッドシート連携スクリプト

2GAS Webhookの設定（スプレッドシート連携）

① スプレッドシートでGASを開く
Googleスプレッドシート → 「拡張機能」→「Apps Script」

② スクリプトを貼り付け
GAS_Webhook受信スクリプト.js の内容を全選択してエディタに貼り付け

③ デプロイ
「デプロイ」→「新しいデプロイ」→「種類：Webアプリ」
アクセス権限: 全員 に設定 → 「デプロイ」

④ URLをコピー
表示された https://script.google.com/macros/s/... をコピー

✅ このURLをWebアプリのトップ画面「送信先URL設定」欄に貼り付けると、回答がスプレッドシートに自動保存されます。

3Webサーバーに置く（公開方法）

以下のいずれかの方法でスタッフがアクセスできるようにしてください：

  • GitHub Pages（無料・おすすめ）— GitHubにファイルをアップしてURLを取得
  • Google Drive + HTMLプレビュー — Driveにアップして共有URLを配布
  • レンタルサーバー / VPS — 既存のサーバーにアップロード
  • Netlify / Vercel（無料）— ドラッグ＆ドロップでデプロイ可能
⚠️ ローカルのhtmlファイルをスタッフに配布する場合、GAS送信機能はCORSエラーで動作しない場合があります。必ずWebサーバー経由で公開してください。

4スマホにインストール（ホーム画面追加）

Android（Chrome）：
URLを開く → アドレスバーの「…」→「ホーム画面に追加」

iPhone（Safari）：
URLを開く → 下の共有ボタン → 「ホーム画面に追加」

✅ ホーム画面に追加するとアプリアイコンが表示され、フルスクリーンで起動できます！

5スタッフ名の変更方法

index.html をテキストエディタで開き、以下の部分を編集してください：

<option>田中太郎（P001）</option>

この行を追加・削除・変更することでスタッフ名一覧を更新できます。
